## 📄 PRD 주요 내용

### 1. 관리자 대시보드 통계

**기능**:
- 총 주문 수, 주문 접수 수, 제조 중 수, 제조 완료 수 표시
- 실시간 업데이트
- 통계 카드 클릭 시 해당 상태의 주문 목록으로 필터링

**API**:
```
GET /api/admin/dashboard/stats
```

**우선순위**: 높음 (Phase 1)

---

### 2. 재고 현황 관리

**기능**:
- 메뉴별 재고 수량 표시
- +/- 버튼으로 재고 조정
- 재고 부족 경고 (재고 < 5개)
- 재고 변경 이력 기록 (선택사항)

**API**:
```
GET /api/admin/inventory
PUT /api/admin/inventory/{menu_id}
```

**데이터베이스**:
- Inventory 테이블 (신규)
- Inventory History 테이블 (선택사항)

**우선순위**: 높음 (Phase 1)

---

### 3. 주문 현황 관리

**기능**:
- 주문 목록 시간순 표시
- 주문 상세 정보 표시 (날짜/시간, 메뉴/수량, 가격)
- 주문 상태 변경 (접수 → 제조중 → 완료)
- 주문 필터링 (상태별, 날짜별)

**API**:
```
GET /api/admin/orders
PUT /api/admin/orders/{order_id}/status
```

**데이터베이스**:
- Order Status History 테이블 (선택사항)

**우선순위**: 높음 (Phase 1)

---

### 4. 메뉴 관리

**기능**:
- 메뉴 등록, 수정, 삭제
- 메뉴 활성화/비활성화
- 이미지 업로드 (선택사항)

**API**:
```
POST /api/admin/menus
PUT /api/admin/menus/{id}
DELETE /api/admin/menus/{id}
```

**우선순위**: 높음 (Phase 1) - 이미 부분 구현됨
